Generic agent GIF one-click sync:

- Put GIFs in this folder and use pattern: <character>_<state>.gif
- States: walking (movement), idle, writing, researching, executing, syncing, error

- GUI mode (recommended):
  python sync_agent_theme.py
  Then click-select each GIF and run.
  Or directly double-click: launch_sync_agent_theme_gui.bat

- Auto-scan mode (example for Songjiang):
  python sync_agent_theme.py --character songjiang --scan

- Backward-compatible quick command:
  python sync_songjiang_theme.py
